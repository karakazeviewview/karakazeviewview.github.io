RPG制作

JavaでテキストベースのRPGを作ろう。
RPGつまりロールプレイングゲームの最初期の形はコンピュータを使わず、ストーリーのシナリオ、イベントや戦闘の成否を判定する計算式とサイコロに鉛筆、コマとなるミニフィギュア、などから成立していた。これらを語り部たるゲームマスターが取りまとめ、何人かの参加者たちは竜を退治する騎士にあるいは肉体改造したドラッグディーラーに自己投影する。
しかし、複数人でなければできない点、またことあるごとに紙と鉛筆とサイコロで計算しなければならない点、これらをコンピュータを使う事によって改良できないか?
ここからコンピュータRPGの歴史が始まり、ローグ、ウィザードリィ、ウルティマ、そして現在の最高峰スカイリムVRなどの名作が生まれた。
では、少しずつ作っていこう。 まず最初に、必要なクラスを作る。今回はキャラクターであるArcherクラスと、そのキャラクターを動かすArcherAppクラスを、2枚のファイルにそれぞれ作ってみよう。 

「1枚目ファイル名:Archer.java」

public class Archer{
	private static int totalArcher;
	private int no;
	public Archer(){
		totalArcher++;
		this.no=totalArcher;
		System.out.println("Archer"+this.no+"を生成しました");
	}

	public static int getTotalArcher(){
		return totalArcher;
	}

	public void shootArrow(){
		System.out.println("Archer"+this.no+"は矢を放った！");
	}
}
